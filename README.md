# notesapp
# 📝 Notes App (Flask + SQLite)

A simple Notes Application built using Flask and SQLite.  
Users can add, view, and delete notes with a clean UI.

---

## 🚀 Features

- Add new notes
- View all notes
- Delete notes
- Clean UI with Bootstrap
- SQLite database integration

---

## 🛠️ Tech Stack

- Python (Flask)
- SQLite
- HTML, CSS, Bootstrap
